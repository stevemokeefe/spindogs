# Spindogs

## Property Comparison Tool

A web application that allows users to compare available properties and their features.

### Features

- Interactive comparison table showing multiple properties
- Sticky first column for easy reference
- Hover states to highlight selected columns
- Responsive navigation with filtering options
- Bootstrap 5 styling with custom SCSS

### Tech Stack

- HTML5
- SCSS/CSS3
- Bootstrap 5.3
- Parcel bundler
